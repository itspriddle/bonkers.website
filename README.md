# bonkers.website

A shrine to the chaotic joy of the early web. Mobile-first micro-apps that make people stop scrolling, smile, and show a friend.

Each page is a single, absurd, over-the-top experience — button smashers, fictional businesses, maximum chaos.

The web used to be fun. Let's bring that back.
